# README 桌宠白底截图设计

## 目标

将 `docs/images/desktop-pet-preview.png` 的深色背景替换为纯白 `#FFFFFF`，使图片在 GitHub README 的浅色页面中更自然，同时继续真实展示当前 Electron 桌宠。

## 实现方式

使用当前 Electron 渲染器重新捕获桌宠模式，而不是对旧 PNG 抠图或使用生成式模型重绘。截图环境只改变展示背景，不改变运行时桌宠资产与交互代码。

## 保持不变

- 图片尺寸保持 `1280x900`。
- 魔法书、花体英文、钢笔、蝴蝶、光点、书脊光效和底部交互提示保持当前真实运行效果。
- 构图、缩放和元素位置保持不变。
- 文件路径仍为 `docs/images/desktop-pet-preview.png`。
- 视觉文件总数仍为 8 个，授权状态仍为 `license-pending`。

## 不采用

- 不对旧截图进行颜色阈值抠图，避免书本边缘和半透明光效产生暗边。
- 不使用生成式重绘替换背景，避免花体文字、蝴蝶和书本细节发生变化。
- 不把白色背景写入桌宠的透明 Electron 运行窗口；白底仅用于 README 展示截图。

## 验收标准

- 输出是 `1280x900` PNG。
- 图片四角及其余外部背景为纯白 `RGB(255,255,255)`。
- 桌宠主体、光效、蝴蝶、英文题字和“单击展开 · 双击捕获”均完整可见。
- 图片没有透明边角、深色背景残留、明显暗边或裁切。
- 中英文 README 继续引用原图片路径，无需修改图片标签。
- 自动化测试、严格 readiness、依赖审计和 Git 检查通过。
- 最终提交推送到 GitHub，并确认三个 CI 作业成功。
