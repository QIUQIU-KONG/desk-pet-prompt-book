## 截图来源

新版 `docs/images/app-preview.png` 使用真实 Electron 应用生成，保持展开面板
原生的 `1024x700` 窗口尺寸。截图过程使用独立的临时用户数据目录，禁止读取
或展示用户本机的提示词、项目名称、文件路径和其他个人信息。

- 页面中展示三条提示词，体现当前固定分页布局；
- 展示具有代表性的评分、项目和阶段状态，但不展开菜单或弹窗遮挡主界面。

截图中的魔法书需要完整、居中且没有裁切。书本外部继续保留透明区域，使图片
能够适配 GitHub 的浅色和深色主题。

## 产品理念文案

中文 README 的章节标题改为“产品理念”，英文 README 的章节标题改为
“Product Philosophy”。

两个 README 都使用下面这条完整英文理念：

> **Build agents with clarity. Let prompts become systems.**

`README.md` 在英文原句下方紧接这句中文翻译：

> 明确地构建代理，让提示词成为系统。

`README.en.md` 保留英文理念并使用英文解释，不重复放置中文翻译。

补充说明需要明确：书页题字不是随机装饰，而是产品理念的一部分。明确地搭建
Agent，可以让零散提示词经过复用、项目和阶段归纳，逐渐形成稳定、可重复的
工作系统。保留右侧钢笔“把想法写下并转化为可执行工作”的象征，但整体说明
保持凝练。
