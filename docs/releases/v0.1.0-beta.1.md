# 桌宠提示词魔法书 v0.1.0-beta.1

这是第一个可直接安装的 Windows Beta。它用于公开测试桌宠捕获、魔法书面板、提示词整理和本地持久化流程，还不是签名的稳定正式版。

## 下载与安装

1. 下载 `Desk-Pet-Prompt-Book-Setup-0.1.0-beta.1.exe`。
2. 双击安装包，按向导选择当前用户的安装目录并完成安装。
3. 安装器会创建桌面快捷方式和开始菜单（Start menu）快捷方式，并可在完成后立即启动应用。
4. 应用不会自动开机启动（does not start automatically）。

本 Beta 是 unsigned（未签名）安装包。Windows SmartScreen 可能显示“Windows 已保护你的电脑”；确认文件来自本仓库 Release 后，可选择 `更多信息 -> 仍要运行`。无法接受该风险时，请不要运行安装包，可继续从源码审查和运行项目。

可使用同一 Release 中的 `SHA256SUMS.txt` 核对下载文件：

```powershell
Get-FileHash .\Desk-Pet-Prompt-Book-Setup-0.1.0-beta.1.exe -Algorithm SHA256
```

## 数据与卸载

提示词、项目、阶段、关键词和视图状态保存在：

```text
%APPDATA%\desk-pet-prompt-book
```

卸载（uninstall）会移除应用文件及快捷方式，但会保留（retains）该目录中的提示词数据，避免误删用户资料。需要彻底清除数据时，请先退出应用、确认不再需要提示词库，再单独删除此目录。

应用强制单实例运行。右键桌宠或展开面板会显示原生 `退出桌宠` 菜单；选择该命令后应用正常退出。

## 已知限制（Known limitations）

- 安装包没有代码签名，Windows 可能显示 SmartScreen 警告。
- 目前没有自动更新；后续版本需要手动下载和安装。
- 当前以 Windows 10/11 为优先平台。
- 本地 JSON 提示词数据没有静态加密。
- 这是 Beta，不承诺生产稳定性或无缺陷运行。
